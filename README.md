# 数据挖掘

基于位存储运算的实现的 数据流中结构二叉树挖掘算法(AMST)
改进原算法无法通用问题。

# 改进原理
 
基于位运算存储编写数据读取抽象代码,使AMST 算法适用于更多的数据组织性形式.
以及更快的运算速度，更少的空间占用。
